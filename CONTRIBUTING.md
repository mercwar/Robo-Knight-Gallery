## Development Guidelines

1. **AIFVS-ARTIFACT Headers**  
   Required for all gallery entries, metadata files, and descriptive objects.

2. **Canonical Accuracy**  
   All additions must align with established Robo-Knight lore and geometry.

3. **Artifact Integrity**  
   Do not modify existing gallery entries without justification and lineage
   documentation.

## How to Submit

- Fork the repository  
- Create a feature branch (`git checkout -b feature/RK-Gallery-Update`)  
- Submit a Pull Request describing the archival addition  

*Gallery Law: Every artifact must preserve the lineage of the Knight.*
